# C-String-Library

Учебная библиотека на языке C для работы со строками, реализующая собственную структуру String и набор функций для безопасной обработки текстовых данных.

---

## Описание проекта

Проект представляет собой реализацию кастомной строковой библиотеки на C.  
Вместо стандартных char* используется собственная структура String, что позволяет:

- контролировать выделение памяти  
- отслеживать длину строк  
- безопасно выполнять операции копирования, вставки и поиска  

В проекте также реализована система обработки ошибок и набор тестов для проверки корректности работы функций.

---

## Функциональность

- Инициализация и освобождение строк  
- Копирование строк  
- Конкатенация  
- Вставка подстроки  
- Поиск подстроки  
- Работа с массивом строк  
- Обработка ошибок  
- Тестирование всех функций  

---

## Структура проекта

├── main.c            # Тестирование функций  
├── string_utils.c    # Реализация строковых операций  
└── string_utils.h    # Заголовочный файл и структура String  

---

## Основные возможности

- Собственная структура String  
- Динамическое управление памятью  
- Проверки на переполнение буфера  
- Централизованная система ошибок  
- Поддержка массивов строк  
- Подробные тесты в main.c  

---

## Сборка и запуск

gcc -Wall -Wextra -std=c11 main.c string_utils.c -o string_lib  
./string_lib  

---

## Пример использования

String s1, s2;

initString(&s1, "Hello");  
initString(&s2, "World");  

concatString(&s1, &s2);  
printstr(&s1);  

freeString(&s1);  
freeString(&s2);
